# StarCraft 2 Auto Replay Fixer (星际2录像全自动修复助手)

这是一个 Python 脚本，用于自动监控并修复星际争霸2（国服版本）无法播放的录像文件。

## ✨ 功能特点 (Features)
- **全自动监控**：无需手动操作，脚本在后台运行。
- **全域搜索**：自动寻找 `StarCraft II` 文件夹，包含所有子目录。
- **不破坏原文件**：修复后的文件会另存为 `_fixed.SC2Replay`。
- **双语提示**：支持中英双语日志显示。

## 🛠️ 如何使用 (How to use)
1. 确保电脑已安装 [Python 3](https://www.python.org/)。
2. 下载本仓库中的 `AutoFix.py`。
3. 双击运行即可。
